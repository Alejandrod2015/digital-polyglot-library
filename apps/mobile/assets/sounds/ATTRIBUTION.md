## If you want to replace any of these

Recommended CC0 sources:

- https://kenney.nl/assets — game-oriented packs, all CC0
- https://freesound.org — filter by `License: CC0 1.0` before downloading
- https://mixkit.co/free-sound-effects/ — Mixkit License (royalty-free, commercial OK, no attribution)

Keep the same filename and 44.1 kHz / mono / ~50 KB or less so the
in-app caching and prefetch behavior stays consistent.
